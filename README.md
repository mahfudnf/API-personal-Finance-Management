



# API personal finance management

A backend application built with Java Spring Boot.
This project features:
- User management
- Income management
- Expense management
- Savings management
- Dashboard display

## 📖 API Reference

To read the API document, please go to the following link Swagger UI:
```text
    http://localhost:8080/swagger-ui.html
```

Note: Run the application locally to access the documentation.

## 🔨 Tech Stack

- Java 17
- Spring Boot
- MySql
- Maven

## ⚙️ Environment Configuration

This project uses **environment variables** for configuration. 

### Step 1 — Create `.env` file

Copy the environment template:

```bash
cp .env.example .env
```

### Step 2 — Fill in required variables

Edit `.env` according to your local setup:

```env
DB_URL=your_db_url_here
DB_USERNAME=your_db_username_here
DB_PASSWORD=your_db_password_here
```

> `.env.example` is provided as a reference.  
> `.env` is intentionally excluded from version control for security reasons.
